# Teslo API

1. Git clone ...
2. Execute

```
yarn install
```

3. Have Nest CLI installed

```
npm i -g @nestjs/cli
```

4. Up database

```
docker-compose up -d
```

5. Clone **.env.template** file and rename it to **.env**

6. fill the environment variables defined in the **.env**

7. Run API

```
yarn start:dev
```
